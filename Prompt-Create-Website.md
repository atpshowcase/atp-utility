# Website Creation Prompt

## Project Goal
Create a comprehensive utility web application similar to "iLovePDF" or "iLoveIMG". The platform should offer various tools for file manipulation, processing, and conversion.

## Technology Stack
- **Frontend**: Next.js (React), styled with modern CSS/Tailwind for a premium, responsive UI.
- **Backend**: Golang (Go) for high-performance API processing.
- **Database**: PostgreSQL for storing user data, tool usage metrics, and configurations.

## Core Features
1. **Landing Page**: A modern, dynamic, and visually stunning landing page showcasing the available tools.
2. **Utility Modules**:
   - **PDF Tools**: Merge, split, compress, convert to/from PDF.
   - **Image Tools**: Resize, crop, compress, convert formats.
3. **User Experience**: Fast processing, clear error handling, drag-and-drop file upload zones, and micro-animations for feedback.
